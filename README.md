# Guns n Hoses — Pressure Washing Website

Premium multipage marketing site built with **Vite + React 18 + TypeScript + React Router DOM v6 + Tailwind CSS v3 + Framer Motion**.

## Run locally
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Deploy to Vercel
1. Push this folder to a GitHub repo.
2. Import on https://vercel.com/new — Vercel auto-detects Vite.
3. SPA refresh works thanks to `vercel.json` rewrites.

## Stack
- React 18 / Vite 5 / TypeScript
- React Router DOM v6 (BrowserRouter)
- Tailwind CSS v3 + tailwindcss-animate
- Framer Motion
- Lucide React icons
